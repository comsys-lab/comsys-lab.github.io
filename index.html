Welcome to ComSys!

ComSys is pioneering researches on various topics in computer architecture and system software designs. We design innovative computing systems to achieve better energy efficiencies and throughput in the Post Moore era.

Join Us!

Comsys is actively recruiting graduate students. Please contact Prof. Yunho Oh if you are interested in research on the following areas.

- Processor architectures (CPUs, GPUs, and DNN accelerators)
- Memory systems (Virtual memory, SSD, and heterogeneous memory)
- Processing-In-Memory (PIM)
- System software (Operating systems, firmwares)
- Parallel programming for CPUs, GPUs, and DNN accelerators
